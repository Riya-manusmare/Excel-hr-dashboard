HR Dataset Analysis & Visualization Dashboard
Project Overview

This project presents an interactive HR Analytics Dashboard developed using Microsoft Excel to analyze workforce data and deliver actionable insights for HR and management teams.
The dashboard focuses on employee demographics, attrition trends, salary distribution, absences, recruitment sources, and overall workforce performance.

Tools & Technologies

Microsoft Excel

Pivot Tables and Pivot Charts

Slicers for interactivity

Excel formulas for data transformation

Dashboard design and visualization techniques

Dataset Summary

The dataset contains employee-level information including department, gender, age, years of experience, salary, absences, attrition status, recruitment source, manager name, and marital status.

Key Metrics (KPIs)

Total Employees: 311

Current Employees: 207

Old Employees: 104

Total Departments: 6

Average Age: 46 years

Average Experience: 12 years

Average Salary: 69,021

Attrition Rate: 0.33

Dashboard Insights

Department-wise employee distribution highlights Production as the largest department.

Gender and marital status analysis provides workforce demographic insights.

Department-wise salary trends identify compensation differences across roles.

Absence analysis helps detect departments with higher absenteeism.

Recruitment source analysis evaluates hiring effectiveness.

Attrition analysis by department, manager, and marital status identifies high-risk areas.

Employee satisfaction levels are compared across departments.

Interactive Features

The dashboard includes slicers for:

Gender

Years of experience

Department

Manager name

These filters allow users to analyze specific employee segments dynamically.
